Когда использовать:
    * Есть зависимые группы объектов (в нашем примере - разные классы юнитов различающихся в зависимости от уровня игрока)
    * Высока вероятность добавления новых групп
Общий интерфейс (интерфейс фабрики) для создания объектов, разные реализации методов создания в разных реализациях фабрики.
